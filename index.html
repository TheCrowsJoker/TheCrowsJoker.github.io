About

My name is Liam Sharpe.

I have a keen interest in computer science and one day hope to be a computer programmer.

I have programming skills in many languages including; python, HTML, CSS, and PHP. I also have experience working with linux servers, and I work as an intern at Catalyst IT as a front-end developer.

Projects

Catalyst IT internship

Since May of 2015, I have been working at Catalyst IT as a front end developer. The main project I work on is called Mahara. It is an open source ePortfolio used by many institutes and organisations around the world. Since starting on the project, I have helped work on the newest design overhaul that was released in October and have made several custom themes for clients.

Catalyst academy / Piwik theme

Over the 2015 summer holidays, I participated in the Catalyst IT open source academy. While there, I learnt a whole lot of new things including programming languages (python, HTML, CSS, PHP, and javascript), testing, version control, graphic design, accessibility, linux, servers, and using the famous vi/vim. I also got to contribute to Piwik - an open source web analytics program - where I created the first theme which is available on the piwik marketplace here.
